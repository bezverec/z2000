# Documentation Archive

This directory preserves completed campaign plans and superseded planning
snapshots. They are implementation history, not the current work queue.

Use these active documents instead:

- [`../roadmap.md`](../roadmap.md) for strategic direction and profile policy;
- [`../next_steps.md`](../next_steps.md) for the ordered implementation queue;
- [`../optimization_plan.md`](../optimization_plan.md) for the measured
  performance campaign;
- [`../iso_coverage.md`](../iso_coverage.md) for the current coverage score.

Archived files retain their original dates and status language. Statements
such as "next", "unsupported", or "planned" describe the repository at that
historical checkpoint and may no longer be current.
